# Radical Rampage

By Loch And Loaded: Digital

Radical Rampage is a platformer with a high focus on acessibility and fun
We have support for remappable controls, Controller support as well as color blind settings

currently as of the current build we have support for 1 level a tutorial and a custom spray maker with other levels in the works
